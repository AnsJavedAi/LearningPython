# 🚀 Mini Project

## 🎮 Number Guessing Game

A command-line game where the computer randomly selects a number, and the player attempts to guess it.

### Features

* Random number generation
* User input handling
* Guess validation
* Hint system ("Too High" / "Too Low")
* Unlimited attempts until the correct guess
* Success message displaying total attempts
