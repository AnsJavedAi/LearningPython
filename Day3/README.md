# 🚀 Mini Project

## 🔐 Password Strength Checker

A command-line application that evaluates the strength of a user's password based on common security rules.

### Features

* User password input
* Password length validation
* Uppercase letter detection
* Lowercase letter detection
* Digit detection
* Special character detection
* Strength rating:

  * Weak
  * Medium
  * Strong
* User-friendly feedback for improving password security
