# 🐍 Learning Python

> My structured 9-Day Python learning journey — documenting concepts, practice problems, and mini-projects while building a strong foundation for Artificial Intelligence and Machine Learning.

---

## 📌 About

This repository contains my complete Python learning journey from the fundamentals to object-oriented programming.

Instead of only studying theory, each day includes:

* 📚 Learning core concepts
* 💻 Solving practice problems
* 🚀 Building a mini-project
* 📝 Writing notes and reviewing concepts

The goal is not only to learn Python syntax, but also to develop strong programming and problem-solving skills.
